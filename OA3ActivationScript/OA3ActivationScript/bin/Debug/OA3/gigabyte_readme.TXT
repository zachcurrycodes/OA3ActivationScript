Usage :
       1.WinOA30_64.exe /f filename : Inject OA3.0 key from file.
       2.WinOA30_64.exe /c          : Clear the OA3.0 key. 
       3.WinOA30_64.exe /h          : Usage
       4.WinOA30_64.exe /i filename : Inject OEM ID from file.
       
 Note : 1 and 4 可以合併使用 ex: WinOA30_64.exe /f filename1 /i filename2
